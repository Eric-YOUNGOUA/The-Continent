1. Overview

This project is a double page website designed to introduce visitors to Cameroon. It aims to provide key information about the country's location and borders, showcase its diverse tourist attractions, explain its official and common languages (including Camfranglais), and present some of its well-known celebrities. The site also includes an interactive quiz to test visitors' knowledge.

---

2. Features

Responsive Navigation: A sticky header with navigation links that smoothly scroll to different sections of the page.
Cameroon Introduction: A dedicated section detailing the country's geographical context, capital, major city, and official languages.
Tourism Showcase: Highlights prominent tourist destinations with images and descriptions, using a responsive layout that alternates image and text placement.
Language Insights: Explains the linguistic landscape of Cameroon, including a table of Camfranglais words and an example sentence.
Celebrity Spotlight: Features a grid of notable Cameroonians with their images and names.
Interactive Quiz: A multi-question quiz covering information presented on the page, with instant scoring upon submission.
Dynamic Elements:
    * Header `<h1>` changes color on hover.
    * Random celebrity image display (requires additional HTML button and image element).
Contact Information: A footer containing contact details and a link for user impressions.

---

3. Technologies Used

HTML5: For the core structure and content of the web page.
CSS3: For styling and layout, including responsive design elements.
JavaScript: For interactive features like smooth scrolling, form validation, and dynamic content manipulation.

---

4. Setup Instructions

To view and run this project locally:

1.  Clone the repository or download the project files to your local machine.
2.  Navigate to the project directory in your file explorer.
3.  Open the `index.html` file in your preferred web browser (e.g., Chrome, Firefox, Edge).

No special server setup or dependencies are required, as it's a static website.
